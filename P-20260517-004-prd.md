# PRD: 卡牌DBG V21 — 遗物(Relic)系统

## 1. 背景与目标

### 背景
V20已实现26个融合配方和稀有度系统。V21参考trading-agents-design Portfolio Manager的资产配置思想，新增**遗物(Relic)系统**：战斗前可装备1-3个遗物，提供被动效果。

### 目标
- 新增遗物系统：被动道具影响战斗策略
- 遗物获取：击败敌人概率掉落
- 遗物装备：战斗前选择装备（最多3个）
- 与现有系统集成：与PhaseEngine、CardHooks协同

### 2.2 遗物装备系统

```
开始游戏
  ↓
关卡1开始前 → 遗物选择界面（最多3个）
  ↓
战斗 → 遗物被动效果生效
  ↓
击败敌人 → 掉落遗物（概率）
  ↓
关卡2开始前 → 重新选择遗物
  ↓
...直到通关
```

### 2.3 遗物效果触发时机

| 遗物类型 | 触发时机 | 实现方式 |
|----------|----------|----------|
| attack | 攻击时 | 在playCard()的attack分支叠加value |
| crit | 攻击时 | 随机判定，暴击时伤害×2 |
| block | 每回合开始 | TurnEndPhase→DrawPhase时自动添加 |
| hp | 战斗开始 | startGame()时增加maxHp |
| energy | 每回合开始 | DrawPhase开始时增加energy |
| lifesteal | 攻击造成伤害后 | 伤害值的value%转化为HP |
| dodge | 敌人攻击时 | 随机判定，躲避则不掉血 |
| burn | 攻击时 | 攻击后附加burn状态（每回合掉血） |

## 4. 技术方案

### 4.1 新增代码结构

```javascript
// ==================== RELIC SYSTEM ====================
const RELIC_TEMPLATES = { ... };

class Relic {
  constructor(name, template) {
    this.name = name;
    this.type = template.type;
    this.value = template.value;
    this.desc = template.desc;
    this.rarity = template.rarity;
  }
}

// RelicManager
const RelicManager = {
  owned: [],        // 拥有的遗物列表
  equipped: [],     // 当前装备的遗物（最多3个）
  
  addRelic(relic) { ... },
  equipRelic(relic) { ... },
  unequipRelic(relic) { ... },
  getEquipped() { return this.equipped; },
  
  // 效果触发
  onAttack(card, damage) { ... },     // attack/crit/lifesteal
  onTurnStart() { ... },              // block/energy
  onEnemyAttack(damage) { ... },      // dodge
  onPlayCard(card) { ... },           // burn
};

// 掉落系统
function tryDropRelic() {
  if (Math.random() < 0.3) {
    const templates = Object.keys(RELIC_TEMPLATES);
    const name = templates[Math.floor(Math.random() * templates.length)];
    const relic = new Relic(name, RELIC_TEMPLATES[name]);
    RelicManager.addRelic(relic);
    showMessage(`🎁 获得遗物：${name}！`);
  }
}
```

### 4.2 遗物选择界面

在deck-modal旁边新增relic-modal：
- 显示所有拥有的遗物
- 点击切换装备状态
- 底部显示"已装备(0/3)"
- 确认按钮进入战斗

### 4.3 效果集成点

- `onAttack()`: playCard()中attack分支调用
- `onTurnStart()`: PhaseEngine.DrawPhase.enter()调用
- `onEnemyAttack()`: executeEnemyIntent()中伤害计算前调用
- `onPlayCard()`: playCard()最后调用（处理burn等）
