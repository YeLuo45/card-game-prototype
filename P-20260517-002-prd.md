# PRD: 卡牌DBG V19 — 卡牌融合系统

## 1. 背景与目标

### 背景
V18已实现PhaseEngine战斗阶段工作流引擎，卡牌效果全部Hook化。V19基于此架构新增**卡牌融合**机制：玩家可将手牌/牌组中的两张卡牌融合，生成一张高阶新卡。

参考nanobot-design的ToolRegistry插件化思想：融合配方如同工具注册表，输入→输出明确可扩展。

### 目标
- 新增卡牌融合玩法，增加策略深度
- 融合配方可扩展（类似ToolRegistry的注册机制）
- 与阶段引擎集成：融合操作可在PlayerActionPhase进行

| 配方 | 输入 | 输出 | 效果 |

### 2.3 融合流程

```
1. 玩家点击"融合"按钮（PlayerActionPhase可用）
2. 进入融合模式：选择第一张卡
3. 选择第二张卡
4. FusionRegistry查找配方
5. 若配方存在：
   - 两张卡从手牌/牌组移除
   - 新卡加入手牌（不可升级）
   - 消耗能量（当前能量的一半，向下取整，最低0）
6. 若配方不存在：提示"无法融合"
```
