# 卡牌DBG - 斩杀史莱姆

卡牌构筑游戏（DBG）原型，验证核心战斗循环和卡牌构筑机制。

## 访问地址

https://yeluo45.github.io/card-game-prototype/

## 项目说明

基于杀戮尖塔（Slay the Spire）玩法的卡牌战斗原型，验证以下核心系统：
- 回合制战斗流程
- 能量系统
- 卡牌收集与构筑
- 敌人AI意图系统

## 功能特性

- 回合制战斗：抽5张牌 → 出牌 → 结算 → 敌人行动
- 能量系统：每回合3点能量，卡牌消耗能量出牌
- 护盾机制：防御牌获得护盾，可抵挡伤害
- 敌人意图：每回合显示敌人意图（攻击/防御/强化）
- 牌堆管理：抽牌堆、弃牌堆，弃牌堆耗尽自动洗牌

## 初始卡组

| 卡牌 | 费用 | 类型 | 效果 |
|------|------|------|------|
| 打击 x5 | 1 | 攻击 | 造成6点伤害 |
| 防御 x5 | 1 | 技能 | 获得5点护盾 |
| 重击 | 2 | 攻击 | 造成12点伤害 |
| 重拳 | 2 | 攻击 | 造成8点伤害 |
| 硬撑 | 1 | 技能 | 获得8点护盾，抽1张牌 |

## 技术栈

- HTML5 + CSS3 + Vanilla JavaScript
- 无框架依赖，纯前端静态页面
- Canvas/HTML渲染
