# DBG 卡牌游戏 V3

基于 DBG（Deck-Building Game）机制的卡牌战斗游戏。

## 游戏地址
https://yeluo45.github.io/card-game-prototype/

## V3 新增功能

### 地图/进度系统
- **3层楼地图**：每层有多个节点（战斗/精英/休息/商店/Boss）
- **节点连接选择**：完成后解锁下一节点
- **跨战斗资源管理**：生命值跨战斗保留，金币可用于商店

### 节点类型
| 节点 | 图标 | 说明 |
|------|------|------|
| 战斗 | ⚔️ | 普通敌人战斗 |
| 精英战 | 👹 | 高难度战斗，胜利后获得金币奖励 |
| Boss战 | 💀 | 每层楼结尾，击败后进入下一层 |
| 休息 | 🏠 | 恢复15点生命 |
| 商店 | 🛒 | 花金币购买卡牌 |

### 楼层结构
- **第1层（5节点）**：休息→战斗→战斗→精英战→商店→Boss
- **第2层（6节点）**：商店→精英战→战斗→战斗→休息→战斗→Boss
- **第3层（7节点）**：休息→精英战→战斗→战斗→商店→战斗→战斗→Boss

### 敌人配置
| 楼层 | 普通敌人 | 精英敌人 | Boss |
|------|----------|----------|------|
| 1 | 史莱姆、哥布林 | 骷髅战士 | 巨石像(30血) |
| 2 | 哥布林、骷髅战士 | 火焰精灵 | 巨石像(45血) |
| 3 | 骷髅战士、火焰精灵 | 骷髅战士 | 巨石像(60血) |

### 游戏结束条件
- **胜利**：击败第3层Boss
- **失败**：生命值归零

## V2 功能（保留）

### 卡牌系统
- **初始牌组**：打击x4、防御x4、硬撑x2、连续打击x2、痛击x1（共13张）
- **卡牌类型**：攻击牌、技能牌、能力牌
- **新卡牌池**：斩首、铁壁、急救、战吼、反击姿态、尖刺外壳、灼热拳头

### Debuff 系统
- **中毒**：每回合3点伤害，可叠加，持续3回合
- **灼热**：每回合2点伤害，可叠加，持续2回合
- **易伤**：受到伤害+50%，不可叠加，持续2回合
- **虚弱**：攻击力-50%，不可叠加，持续2回合

### 敌人系统
- **史莱姆**（20血）、**哥布林**（28血）、**骷髅战士**（35血）、**火焰精灵**（25血）
- **巨石像Boss**（50血+10护甲，每3回合获得8护盾）

### 核心机制
- 能量：3点/回合
- 抽牌：回合开始抽5张
- 护盾：优先抵消伤害
- 战斗胜利后选牌奖励

## 技术栈
- 单文件 HTML
- Vanilla JS（无框架）
- Canvas 绘制地图
- CSS3 动画

## 本地运行
直接用浏览器打开 `index.html` 即可。
