# DBG 卡牌游戏 V2

基于 DBG（Deck-Building Game）机制的卡牌战斗游戏。

## 游戏地址
https://yeluo45.github.io/card-game-prototype/

## V2 新增功能

### 卡牌系统
- **初始牌组调整**：打击x4、防御x4、硬撑x2、连续打击x2、痛击x1（共13张）
- **V2新增卡牌**：
  - 攻击牌：连续打击、痛击、斩首
  - 技能牌：铁壁、急救、战吼
  - 能力牌：反击姿态、尖刺外壳、灼热拳头

### Debuff 系统
- **中毒**：每回合3点伤害，可叠加，持续3回合
- **灼热**：每回合2点伤害，可叠加，持续2回合
- **易伤**：受到伤害+50%，不可叠加，持续2回合
- **虚弱**：攻击力-50%，不可叠加，持续2回合
- **荆棘**：反弹伤害

### 敌人系统
- **史莱姆**（保留）：20血，随机攻击/防御意图
- **哥布林**：28血，高攻速（小伤害多次）
- **骷髅战士**：35血，高防御（常驻护盾）
- **火焰精灵**：25血，频繁使用灼热debuff
- **巨石像**（Boss）：50血+10护甲，每3回合获得8护盾

### 意图系统
- 攻击：造成X点伤害
- 防御：获得X点护盾
- 强化：提升攻击力/施加debuff
- 特殊：Boss专属重压（全体攻击）

### UI 改进
- 右侧战斗日志面板（最新5条）
- 状态图标可视化（debuff图标）
- 卡牌悬停显示完整描述
- 战斗胜利后选牌奖励

## 核心机制
- 能量：3点/回合
- 抽牌：回合开始抽5张
- 护盾：优先抵消伤害
- 回合结束：结算debuff和荆棘反伤

## 技术栈
- 单文件 HTML
- Vanilla JS（无框架）
- CSS3 动画

## 本地运行
直接用浏览器打开 `index.html` 即可。
