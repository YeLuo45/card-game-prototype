# DBG 卡牌游戏 V5

基于 DBG（Deck-Building Game）机制的卡牌战斗游戏。

## 游戏地址
https://yeluo45.github.io/card-game-prototype/

## V5 新增功能

### 战斗奖励卡牌选择系统
- **分层奖励池**：普通奖励池和稀有奖励池
- **精英/Boss战稀有奖励**：精英战斗和Boss战提供稀有奖励池
- **跳过奖励功能**：玩家可选择跳过卡牌奖励
- **增强UI**：清晰的奖励选择界面，带跳过按钮

### 奖励池配置
| 战斗类型 | 奖励数量 | 奖励池 |
|---------|---------|--------|
| 普通战斗 | 2张 | 普通池 |
| 精英/Boss战 | 3张 | 稀有池 |

### 普通奖励池
- 连续打击、铁壁、急救

### 稀有奖励池
- 痛击、斩首、战吼、反击姿态、尖刺外壳、灼热拳头
- 恶魔契约、闪电动作、冥想、致命精准、吸血

## V4 功能（保留）

### 地图/进度系统
- **3层楼地图**：每层有多个节点（战斗/精英/休息/商店/Boss）
- **节点连接选择**：完成后解锁下一节点
- **跨战斗资源管理**：生命值跨战斗保留，金币可用于商店

### 节点类型
| 节点 | 图标 | 说明 |
|------|------|------|
| 战斗 | ⚔️ | 普通敌人战斗 |
| 精英战 | 👹 | 高难度战斗，胜利后获得金币奖励 |
| Boss战 | 💀 | 每层楼结尾，击败后进入下一层 |
| 休息 | 🏠 | 恢复15点生命 |
| 商店 | 🛒 | 花金币购买卡牌 |

### 诅咒牌与特殊牌系统
- **诅咒牌**：痛苦诅咒、虚弱诅咒、黑暗诅咒、死亡诅咒
- **特殊牌**：恶魔契约、闪电动作、冥想、致命精准、吸血

### 中毒/流血系统
- **中毒**：每回合3点伤害，可叠加，持续3回合
- **流血**：回合开始2点伤害，持续2回合，可叠加

## V3 功能（保留）

### 敌人系统
- **史莱姆**（20血）、**哥布林**（28血）、**骷髅战士**（35血）、**火焰精灵**（25血）
- **巨石像Boss**（50血+10护甲，每3回合获得8护盾）

### 核心机制
- 能量：3点/回合
- 抽牌：回合开始抽5张
- 护盾：优先抵消伤害
- Debuff系统：易伤、虚弱、中毒、灼热、流血

## 技术栈
- 单文件 HTML
- Vanilla JS（无框架）
- Canvas 绘制地图
- CSS3 动画

## 本地运行
直接用浏览器打开 `index.html` 即可。
