# PRD: 卡牌DBG V20 — 融合配方扩展 + 稀有融合卡 + 动画特效

## 1. 背景与目标

### 背景
V19已实现基础融合系统（10个配方），但配方数量有限、融合卡无稀有度区分、缺乏视觉反馈。

### 目标
- 新增10个高级融合配方（需要特定卡牌组合）
- 引入**稀有度系统**：普通/优秀/史诗/传说（影响融合概率和卡牌强度）
- 添加**融合动画特效**：卡牌消失→能量汇聚→新卡生成

### 2.3 融合动画序列

```
1. 选择第二张卡后，显示"融合中..."提示
2. 第一张卡：淡出 + 向第二张卡移动
3. 第二张卡：淡出 + 缩小
4. 能量粒子效果：从中心向外扩散（CSS animation）
5. 新卡从中心放大出现
6. 显示稀有度角标
7. 新卡飘入玩家手牌区域
```

## 4. 技术方案

### 4.1 新增代码

```javascript
// ==================== RARITY SYSTEM ====================
const RARITY = {
  NORMAL: 'normal',
  FINE: 'fine',
  EPIC: 'epic',
  LEGENDARY: 'legendary'
};

const RARITY_COLORS = {
  normal: '#aaa',
  fine: '#4a90d9',
  epic: '#9b59b6',
  legendary: '#f39c12'
};

const RARITY_LABELS = {
  normal: '',
  fine: '✦',
  epic: '◆',
  legendary: '★'
};

// 融合卡稀有度继承：产物稀有度由配方决定
// 融合动画CSS
@keyframes fusion-flash {
  0% { opacity: 1; transform: scale(1); }
  50% { opacity: 0.5; transform: scale(0.8); }
  100% { opacity: 0; transform: scale(0.3); }
}

@keyframes fusion-appear {
  0% { opacity: 0; transform: scale(0) rotate(-180deg); }
  60% { opacity: 1; transform: scale(1.2) rotate(10deg); }
  100% { opacity: 1; transform: scale(1) rotate(0deg); }
}
```

### 4.2 配方注册示例

```javascript
// 龙息 - 史诗品质
fusionRegistry.register(['剑气', '猛击'], {
  name: '烈焰龙息', cost: 3, type: 'attack', value: 25,
  description: '造成25点伤害', rarity: RARITY.EPIC
}, 2);
```
